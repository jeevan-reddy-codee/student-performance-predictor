 # 🎓 Student Performance Predictor

A web-based Machine Learning project that predicts student exam scores based on study hours.

## 🚀 Features
- Predict score using Linear Regression
- Interactive UI with slider
- Real-time prediction using Flask API

## 🛠️ Tech Stack
- Python
- Flask
- HTML / CSS / JavaScript
- Scikit-learn

## 📊 How it works
1. User enters study hours
2. Data sent to Flask backend
3. ML model predicts score
4. Result displayed on UI

## 📷 Output Example
- Input: 9 hours
- Output: 91.13/100
